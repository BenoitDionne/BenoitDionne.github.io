Managing Questions and Assignments

Introduction

If you have created your own questions and assignments in a previous course and would like to use them in your new course, you have to export them from the old course to a zip file and, after that, import this zip file to your new course. We explain how to do it below.

Exporting Questions and Assignments to a zip File

We explain how to export assignments to a zip file. The procedure is almost identical for questions. The only difference is to select questions instead of assignments.

  1. In the course where are located the assignments that you will like to use in your new course, select Content Repository in the top menu (item 1 in the figure below) followed by Assignments (item 2 in the figure below). Then, select the folder where your assignments can be found. Normally, it will be in the folder Default Unit (item 3 in the figure below) but it could be any folder.

  2. Then, click the boxes in front of the assignments that you want to export to a zip file. In the example above (item 4 in the figure above), we selected all the assignments in the folder.

  3. Finally, click on the button Export at the bottom of the window (item 5 in the figure above) to export the selected assignments to a zip file. Your file manager will probably ask you to confirm that you accept to download the zip file to your system. By default, the name of the zip file will be the name of the assignment if you export only one assignment or Coursemodule.zip if you export more than one assignment.

Importing a zip File with Questions and Assignments to a course

  1. On the new course, you select Content Repository in the top menu (item 6 in the figure below). Click on Import at the top left-corner of the new display (item 7 in the figure below).

    You will get the following popup window to select a file on your computer.

  2. Click on Browse in this popup window (item 8 in the figure above) to access the file manager of your computer. After you have selected the zip file that you want to import, you will go back to the popup window. This time, the name of the zip file that you want to import appear on the left of the Browse button (item 9 in the figure below). Click on Import Content on the bottom right corner of the popup window (item 10 in the figure below) to import the zip file.

    You will get the following popup window confirming that the transfer has been done successfully if you have selected a proper zip file.

  3. You may select View the imported content (item 11 in the figure above) or click on Done (item 12 in the figure above).

    The assignments that you have imported will be found in the subfolder Default Unit of the Assignments folder of the course (item 13 in the figure below).

All the questions used for the assignments that you have imported can be found in the folder Questions. They are the questions that you have to edit if ever you need to edit the questions of the assignments that you have imported. If your assignments were using questions from the database of questions, the imported assignments will not use them but will use the imported questions that you can find in the folder Questions.

You can now edit the assignments: change the questions, modify the access and due dates, etc.
