Proctor Tools

Introduction

It may happen that you need to set special accessibility conditions to an assignment for some students. For instance, students with SASS may need extra time, or you may need to restart an assignment for some students. To do all that, you need to use the Proctor Tools (item 1 in the figure below).

Procedure

  1. After clicking on the item Proctor Tools of the upper menu (item 1 in the figure above), you get the following form.

  2. It is possible to search the list of students registered to the course using the Search field but that is not really convenient. You should select Advanced Search (item 2 in the figure above) to get a more detailed form to search not only for students but also for specific assignments.

    The item Roster Search (item 2.5 in the figures above) will be presented in the next section.

  3. There are many fields on this form which are standard: Last Name, First Name, User Login, Student ID and Email. If you have some of this information, you may enter it to go directly to the student that you are looking for. If you plan to modify the accessibility for more than one student, you should ignore those fields. Associated to the previous fields, you can also adjust the fields Match amd Case Sensitive to specialise even more your search.

    The field User Role is set to Student and Guess. There is no reason to change it.

    The really interesting and useful fields are Records per Page and Assignment Filter (items 3 and 4 in the figure above). If you have 200 students in your class and you are looking for two names at the opposite ends of the alphabet, you should set the Records per Page to 200 or more to get them on the same page.

    If you are interested in a specific assignment, you should select it in the field Assignment Filter. For the present example, we have selected the assignment Devoir 3. After clicking on Search (item 5 in the figure above), you get you class list as represented below.

  4. There are many options to select.

    • In 6 in the figure above, you can select to let some students Start/Re-Enter an assignment or Deny Access to an assignment to some students.
    • You can select to grade an assignment for some students by checking the box in front of Grade (item 7 in the figure above).
    • You can even let some students continue to work an assignment that the have already submitted by checking the box in front of Re-open if already graded (item 8 in the figure above).
    • Finally, you can change the deadline for some students by entering the new date and time in the box beside Extend end date until (item 9 in the figure above).

    To change the end date, you have to click on the calendar (item 9 in the figure above) to get the following popup window.

    In the figure above, we assume that you want to change the end date of Devoir 3 for some students to October 10, 2020, at 3:00 pm.

  5. You can check the box in front of Select All or Set Permission (item 10 in the second to last figure). Or you can check the box at the beginning of the line (item 11 in the second to last figure) for each students to whom the modification should apply as in the figure below.

  6. In the figure above, we have assumed that it is the end date of Devoir 3 for the third student that you want to change. So, you check the box on the third line and click on Authorize (item 12 in the figure above) to complete the procedure. You get the following display.

Roster Search

In the previous section, we ignored the link Roster Search (item 2.5 in the second and third figures of the previous section). It is now time to talk about it. This feature of Proctor Tools is really useful if you are looking for many students for whom you know the User Login.

  1. If you click on the link Roster Search, you get the following form.

  2. You use the Browse button to the right of the item Roster File (item 1 in the figure above) to upload a CVS file with the list of students that you are looking for. The CVS file must have one column with title Login. On each row, you enter the User Login of one student.

    You use the item Assignment Filter/ (item 2 in the figure above) to select the assignment whose accessibility conditions will be modified.

    You should set the Records Per Page (item 3 in the figure above) to a value at least equal to the number of students in your CVS file.

    Suppose that you have done all these modifications to get the following display.

    Namely, you end up at step 4 in the previous section. You can then go on to modify the accessibility conditions of the selected assignment for the students in your CSV file.
