How to Use Möbius

We have divided the instruction material into several topics. The topics that the instructor must read carefully are:

  • : We explain how to preview questions in the bank of questions. This is the first step to create an assignment when the instructors select the questions that they want for their assignments. It suffices to write down the numbers of the questions.
  • : We explain how to create an assignment in Möbius using the list of questions that has been selected previously.
  • : After having selected the questions for the assignment, it is time to set the rules (maximal grade, due date, number of attempts, access to How did I do?, ...). This topic addresses this subject.

The following topics may occasionally be useful.

  • : This type of assignments is unfortunately not often used. Students can get harder or easier questions according to their success in previous questions. The passing grade for an assignment may be determined by successfully answering questions at a certain level. We explain how to create adaptive assignment in this topic.
  • : We explain how to change the due date of an assignment after it is active or to modify the rules of an assignment for a student with special needs.
  • : It occasionally happen that we need to modify the grade of a student assignment. This topic explains how to do it.
  • : This topic is only for those who use Möbius outside BrightSpace. If the Möbius assignments are accessed from a LTI link in BrightSpace, the grades are automatically transferred to the gradebook on BrightSpace and nothing needs to be done.
  • : For the instructors who will create questions in Möbius that they plan to use for many courses, we explain how to move questions from one course to the other.
  • : Sometime, an instructor would like to know who posted one of the questions of an assignment on the website to get some help (i.e. to cheat). We explain how to find out the identity of the student.
