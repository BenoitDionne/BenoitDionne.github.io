Fractal Dimensions

There is no clear definition of a fractal set. Benoit B. Mandelbrot (the father of Fractal Geometry) defines a fractal set as a set for which the Hausdorff dimension strictly exceeds the topological dimension. Some mathematicians will also request that the set had some self-similarity (this is the case for the Sierpinski gasket).

We will study only the notion of similarity dimension and Hausdorff dimension.

Similarity Dimension

The similarity dimension of a set A is defined by the equation

Dimension = log(a) / log(1/s)

where a is the number of pieces similar to the original set and s is the new scaling of the spatial dimensions. For instance:

  • The similarity dimension of a straight line is 1. If s = 1/4, then a = 4 and log(a)/log(1/s) = 1.
  • The similarity dimension of a cube is 3. If s = 1/2, then a = 8 and log(8)/log(2) = log2(8) = 3.
  • The similarity dimension of the Sierpinski gasket is 1.5850. If s = 1/2, then a = 3 and log(3)/log(2)= 1.5850. If s = 1/4, then a = 9 and log(9)/log(4) = log(3)/log(2)=1.5850 (why?).

Hausdorff Dimension

The Hausdorff dimension of a set A is defined by the equation

Dimension = limµ->0 log(N(A,µ)) / log(1/µ) ,

where N(A,µ) is the smallest number of balls of radius µ > 0 needed to cover A.
