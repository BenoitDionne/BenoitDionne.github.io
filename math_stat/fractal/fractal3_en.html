The Discrete Dynamical Systems (DDS)

The simplest IFS are defined by real valued functions defined on the real line. Instead of using several functions of the plane into itself, we use only one function from the real line into itself. Moreover, instead of starting with a set (triangle, ...), we start with a single point of the real line.

Let f be a real valued function defined on the real line. The equation

xn+1 = f(xn)

defines a discrete dynamical system (DDS). Given any xo, we may produce a sequence

x1 = f(xo),   x2 = f(x1),   x3 = f(x2),   ...

We start with the study of the discrete dynamical systems given by updating functions f which are linear functions.

f(x) = m x + b

The parameter m is the slope of the line represented by the function f and b is the y intercept of this line.

What happen to the sequences x0, x1, x2, ... if you change the values of the parameters m and b?

Logistic Map

The simplest non-linear discrete dynamical system is given by the logistic mapping . However, we will see that the behaviour of the sequences produced by iteration of this function is far from simple. The logistic mapping is defined by

fµ(x) = µ x ( 1- x)   ,

where µ is a fixed but arbitrary parameter. We have a family of functions; one for each value of µ.

We chose a positive value for µ and a value xo of x between 0 and 1. For µ between 1 and 3, independently of the value xo that we chose, the iterations

xn+1 = fµ(xn)

for n = 0, 1, 2, 3, ... will converge toward a single point between 0 and 1.

For each values of µ between 1 and 3, the iterations converge toward a single point p between 0 and 1. The point p is such that fµ(p) = p. For this reason, the point p is called a fixed point of fµ.

The convergence of the iterations toward p is a consequence of the Fixed Point Theorem. This theorem says that if f: [a,b] → [a,b] is a continuous function, there is a fixed point p in [a,b]. Moreover, if there exists a positive constant K < 1 such that |f '(x)| < K for all x in [a,b], then any iteration starting with xo in [a,b] will converge to a unique fixed point p in [a,b].

The function fµ:[0,1] → [0,1] has always a fixed point at the origin. With µ > 1 , there is another fixed point p>0. For 1<µ <3, we can find a small interval [a,b] containing the fixed point p where the Fixed Point Theorem can be applied to conclude that any iteration with initial condition xo in [a,b] converges toward p. For instance, if µ = 2, the application above shows that the graph of f2 intersects the line y = x at only one point p>0, it is the fixed point of f2, and that all iterations starting closed to p converge to p.

For some values of µ (values between 3 and 4 are really interesting), the iterations do not converge toward a single point but they do converge toward a more complex set. What happen when µ is chosen close to 4? The iterations xn+1 = fµ(xn) becomes Chaotic . This is another subject to study but we will not do it here.
