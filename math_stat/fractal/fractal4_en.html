Mandelbrot and Julia Sets

We cannot end our little introduction to iterative function systems without mentioning the figure that started all the excitement about computer generated fractal images; namely, the Mandelbrot set .

In the previous page, we have presented some properties of the logistic mapping. More precisely, we have looked at the iterations of the real valued function

fµ(x) = µ x ( 1- x)

for some values of µ. The logical next step is to consider the iterations of functions from the plane to the plane as we have done for IFS's. If we identify the real plane to the complex plane, functions of the real plane into itself can be seen as complex valued functions defined on the complex plane.

The Julia sets are generated by iterations of the simple complex valued functions defined by

fc(z) = z2 + c   ,

where c and z are complex numbers. More precisely, given c, let M be the set of all points z in the complex plane such that the sequence

zo = z,   z1 = fc(zo),   z2 = fc(z1),   z3 = fc(z2), ...

is unbounded. The Julia Set associated to c is the boundary of M.

If a point of the iteration starting with zo = z goes outside the circle of radius r = max{|c|,2} centred at the origin, then the iteration gives an unbounded sequence. Therefore, z is in M. Based on this property, we say that a point z doesn't belong to M if after a chosen number of iterations, the sequence of iterations is still inside the circle of radius r centred at the origin. If you choose a large number of iterations, you theoretically get a more accurate image of the Julia set. Obviously, the quality of the picture is limited by the number of pixels on the display of your computer. In particular, if M is highly disconnected (formed of many little pieces), choosing a large number of iterations may produce an empty image if most of the points in M aren't associated to pixels on your screen.

The Mandelbrot set is set of all points c in the complex plane such that the following sequence is bounded.

c,   c^2 + c,   (c^2 + c)^2 + c,   ((c^2 + c)^2 + c)^2 + c, ...

If a point of the iteration goes outside the circle of radius r defined above, then the sequence c, c^2 + c, c^2 + c)^2 + c, ((c^2 + c)^2 + c)^2 + c, ... is unbounded. Therefore, this value of c isn't in the Mandelbrot set. As for the Julia sets, this is the rule that we have implemented in the application above.

You may zoom in on a region of the Mandelbrot set by clicking with your mouse in the region of interest in the figure.

You can also produce the Julia set associated to a value of the Mandelbrot set by clicking with your mouse on the point of interest in the Mandelbrot set when holding the Alt key. The associated Julia set is displayed in the window for the Julia sets Application above.

The Mandelbrot set is also called the gingerbread man . The Mandelbrot set looks like a gingerbread man cookie, doesn't it? If you zoom in judiciously, you will see smaller and smaller gingerbread men. This is comparable to the self similarity that we have observed in the images generated by IFS's.

There is a beautiful relation between the Mandelbrot set and the Julia sets. The Mandelbrot set is the set of all points c that produce a connected Julia sets (the set can be drawn without lifting the pen though it may take you an infinite among of time to draw it).

There are many fascinating mathematical questions that came up from the study of the Mandelbrot set and Julia sets. Many of them are still unsolved.
